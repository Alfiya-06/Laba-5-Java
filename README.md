### 1. Класс  `Main`

Основной класс программы, реализующий дружелюбный консольный интерфейс для демонстрации всех заданий.
Отвечает за ввод данных, валидацию, создание объектов и вывод результатов.

**Основные методы:**

`inputInt(Scanner)` — ввод целого числа с проверкой на корректность;

`inputNonZeroInt(Scanner)` — ввод целого ненулевого числа;

`readSortedList(Scanner)` — чтение упорядоченного списка целых чисел с проверкой на возрастание;

`isSorted(List<Integer>)` — проверка, отсортирован ли список по возрастанию.


### 2. Класс  `Fraction`

Представляет обыкновенную дробь в виде числителя и знаменателя.
Реализует интерфейс `FractionOperations`.

Автоматически переносит знак в числитель;

Запрещает нулевой знаменатель (выбрасывает `IllegalArgumentException`);

Реализует кеширование вещественного значения при первом вызове `getDoubleValue()`;

Поддерживает стандартные методы `equals()` и `hashCode()` для корректного сравнения;

Возвращает строковое представление в формате числитель/знаменатель.

**Пример:**

`new Fraction(1, 2).getDoubleValue()` → 0.5

### 3. Интерфейс  `FractionOperations`
Определяет контракт для работы с дробями:

`getDoubleValue()` — получение вещественного значения;

`setNumerator(int)` — установка числителя;

`setDenominator(int)` — установка знаменателя.

### 4. Класс  `Cat` и интерфейс `Meowable`

`Meowable` — функциональный интерфейс с одним методом `void meow()`;

`Cat` — реализует `Meowable`, хранит имя и при вызове `meow()` выводит: «Имя: мяу!»;

Имя обязательно, не может быть пустым или null.

### 5. Класс  `MeowCounter`
Декоратор для объектов типа `Meowable`.

Считает количество вызовов метода `meow()`;

При каждом вызове делегирует действие оригинальному объекту и увеличивает счётчик;

Переопределяет `toString()` для вывода: «Кот мяукнул X раза».

**Пример:**

`Cat cat = new Cat("Барсик")`;

`MeowCounter counter = new MeowCounter(cat)`;

`counter.meow();` // → Барсик: мяу!

`System.out.println(counter);` // → Кот мяукнул 1 раза

### 6. Класс `ListMerger`
Выполняет слияние двух упорядоченных списков в один отсортированный список (по возрастанию).

Конструктор принимает два списка `L1` и `L2`;

Метод `mergeIntoFirst()` модифицирует первый список, добавляя в него элементы второго с сохранением порядка;

Переопределён `toString()` для удобного вывода результата.

**Пример:**

`L1` = [1, 3, 5], `L2` = [2, 4, 6] → результат: [1, 2, 3, 4, 5, 6]

### 7. Класс  `RussianLetter`
Подсчитывает уникальные русские буквы в текстовом файле.

Игнорирует регистр («А» и «а» считаются одной буквой);

Учитывает только буквы кириллицы (диапазоны А–Я, а–я);

Использует `LinkedHashSet` для сохранения порядка первого появления букв;

**Пример:**

Текст: «Привет, мир!» → уникальные буквы: [п, р, и, в, е, т, м], количество: 7.

### 8. Класс  `QueueBuilder`
Создаёт «зеркальную» очередь на основе исходного списка:

вначале добавляет все элементы списка в прямом порядке,
затем — в обратном порядке.

**Пример:**

Вход: [1, 2, 3] → очередь: [1, 2, 3, 3, 2, 1]
